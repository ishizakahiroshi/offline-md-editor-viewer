# 変更履歴

## Unreleased

## 0.3.0 - 2026-07-14

### Added

- コマンドパレット（`Ctrl/Cmd+K` または `Ctrl/Cmd+Shift+P`）を追加。主要な操作をキーボードから実行できる。
- 見出しアウトラインパネルを追加。Markdown の見出し間をジャンプできる。
- 集中執筆用の Zen モードを追加（`Esc` で終了）。
- 最後に保存した内容と現在の編集内容を比較する未保存差分ビューを追加。
- 最近使ったファイル一覧を追加（ワンクリックで履歴消去可）。
- ステータスバーに文字数・単語数・推定読了時間・選択文字数を表示するようにした。
- ツールバーに現在開いているファイル名の表示バーを追加。クリックでファイル名をコピーできる。

### Fixed

- 2026年7月のコード監査に基づく堅牢性修正: Windows デスクトップ版で単体ファイルを起動引数/ドロップで開いたときに親フォルダツリーが平坦化していた問題、並行するファイル読み込みの完了順序が逆転して古い内容が表示されうる問題、バイナリファイル拒否時にファイル選択状態が更新されてしまう問題、ブラウザ版のファイル作成で同時に作られた同名ファイルの中身を消しうる問題、ブラウザ版のフォルダコピーが途中で失敗した際に部分コピーの残骸が残る問題、デスクトップ版のフォルダコピーで Windows のジャンクション/リパースポイントを辿ってしまう問題、BOM なしの短い UTF-16 ファイルの判定漏れを修正。
- 複数選択削除の一部失敗時に、最後の成功表示で失敗が隠れていた問題を修正。削除数・失敗数を通知する。
- 開いているファイルを削除したときに、現在ファイル名の表示が残っていた問題を修正。
- 外部リンクブロック時の通知文言を 13 言語すべてで翻訳するようにした（従来は日本語/英語のみ）。

### Changed

- 同梱 Web 依存を更新: DOMPurify 3.4.12（3.4.8 に公開された脆弱性 2 件への対応。本アプリは `setConfig()`/hook を使わず `sanitize()` を単発で呼ぶだけのため到達不能だが、予防的に更新）、marked 18.0.6。
- Tauri を 2.11.5 へ、`plist` 依存を 1.10.0 へ更新。これにより、v0.2.0 で受容していた RUSTSEC-2026-0194 / RUSTSEC-2026-0195（quick-xml）を `plist`/macOS メタデータ経路で解消した。`rfd` の Linux/Wayland ファイルダイアログ経路に残る quick-xml 0.39.4 は Windows ビルドにはコンパイルされないことを `cargo tree --target x86_64-pc-windows-msvc` で確認済みで、到達不能として受容を継続する。
- デスクトップ版の Content Security Policy を強化（`script-src-attr 'none'`、`worker-src 'none'`、`child-src 'none'` などのディレクティブを追加）。

## 0.2.0 - 2026-07-03

### Added

- ブラウザ版の npm 配布を追加: `npx offline-md-editor-viewer`（常用は `npm i -g offline-md-editor-viewer`、Node.js 18+）で、同梱の単一 HTML ブラウザ版を既定のブラウザで開ける。npm パッケージはランチャーのみで、配布の正本は引き続き GitHub Releases。

### Fixed

- 2026年6月のセキュリティ・品質監査に基づく堅牢性修正: デスクトップ版のファイル作成を原子的に変更（同名ファイルが同時に作られた場合に中身を消さない）、巨大ファイルガードの fail-closed 化、フォルダ一覧でのジャンクション/リパースポイント除外の統一（深い列挙と浅い列挙）、別ボリュームへの移動失敗時のエラー文言の明確化、ブラウザ版の検索・置換およびエンコーディング表示のエッジケース修正。
- デスクトップ版のフォルダツリー展開で、フォルダ内にアクセスできないエントリが 1 つあるだけで展開全体が失敗していた問題を修正（該当エントリのみスキップ）。

### Changed

- 未使用の内部デスクトップコマンドを削除（コマンド露出面の縮小）。

## 0.1.1 - 2026-06-05

### Added

- Markdown プレビューのコードブロックを、同梱 GitHub Dark テーマの highlight.js 11.11.1 でシンタックスハイライト表示するようにした。
- リリース前確認をしやすくするため、デモ動画、スクリーンショット、サンプル Markdown ファイルを追加。
- Windows デスクトップアプリが起動時に最大化されるようにした。

### Changed

- 同梱 Web 依存を marked 18.0.5、DOMPurify 3.4.8 に更新。
- Tauri/Rust 依存一式を更新し、デスクトップ第三者ライセンス情報を再生成。
- スタンドアロン配布物と ZIP 内ファイルのハッシュ一致確認など、リリースアセット生成・検証を強化。
- ファイル/フォルダの作成・名前変更で、制御文字、Windows 予約名、末尾のドット/スペースを明示的に拒否するよう入力検証を強化。

### Fixed

- ドットを含むファイル名のリネーム処理を修正。
- 複数選択削除で、選択したフォルダもファイルと一緒に削除し、親子の重複選択を二重カウントしないよう修正。

## 0.1.0 - 2026-05-04

### Added

- 初回公開リリース
- 編集モードにフローティング検索・置換バーを追加。次/前移動、大小文字区別、単語単位、正規表現検索、1件置換、すべて置換に対応。
- 外部リンク設定を追加。`http` / `https` リンクは既定でブロックし、設定を有効にした場合だけブラウザで開く。
- 完全オフラインで動作するローカルファースト Markdown エディタ・ビューア
- ブラウザ版はライブラリをインライン化した単一 HTML ファイルとして配布し、ビルド不要
- Windows デスクトップ版はポータブル ZIP 形式で配布（インストーラなし）
- エディタとライブプレビューを左右に並べて表示、スクロールは同期
- ファイルおよびフォルダを開く機能。直近フォルダはセッション間で記憶
- ダーク / ライトテーマ対応の13言語 UI
- フォルダ一覧をブラウザ版・Windows デスクトップ版の両方で展開可能なツリー表示に対応
- Windows デスクトップ版は CLI 引数で Markdown ファイルを受け取り可能。エクスプローラーの「プログラムから開く」でアプリに関連付けできる
- Windows デスクトップ版はフォルダのドラッグ&ドロップに対応。フォルダをウィンドウ上にドロップするとファイルツリーを開き、単一ファイルをドロップするとそのファイルを開く
- リリースアセットを4ファイル構成で配布。スタンドアロン `offline-md-editor-viewer.html`、スタンドアロン `offline-md-editor-viewer.exe`、ブラウザ版 ZIP、Windows デスクトップポータブル ZIP の4種類。4アセットすべての SHA-256 チェックサムを公開し、ブラウザ ZIP 内の HTML がスタンドアロン HTML と、デスクトップ ZIP 内の exe がスタンドアロン exe とそれぞれ一致することを CI で検証する。
- About ダイアログに「ライセンス本文を表示」リンクを追加。アプリの MIT ライセンス、同梱 Web 依存ライブラリのライセンス（marked / DOMPurify / encoding-japanese / Feather Icons）、および Windows デスクトップ版では exe に静的リンクされた Tauri/Rust クレートすべてのライセンス全文を確認できる。デスクトップ側のライセンス情報は `cargo-about` で `Cargo.lock` から生成。
- Windows デスクトップのリリースビルドはブラウザ版と同一の単一 HTML を埋め込む（ステージングディレクトリと `apps/desktop/src-tauri/tauri.release.conf.json` を使用）。フロントエンドレイヤーで両版が完全に同一になる。
