# Project Plan

## Goal

Make an offline-first Markdown editor/viewer that feels safe to use with local notes.

## Principles

- Local files stay local
- No cloud account required
- No CDN dependency
- Browser edition first, portable desktop edition when needed

## Milestones

| Milestone | Status |
| --- | --- |
| Browser single HTML | Done |
| Windows portable app | Done |
| Demo video | In progress |
| macOS/Linux investigation | Later |

## Release Note Draft

Offline Markdown editing with live preview, folder browsing, and local-only privacy defaults.
